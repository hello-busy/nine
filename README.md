# Nine

A Python program that explores the fascinating mathematical properties of the number 9.

## Features

- **Multiplication Table**: Display the multiplication table for 9
- **Digital Root Property**: Demonstrate how all multiples of 9 have a digital root of 9
- **Interesting Facts**: Share fascinating facts about the number 9
- **Divisibility Test**: Show how to test if any number is divisible by 9

## Usage

Run the program directly:

```bash
python3 nine.py
```

## What You'll Learn

The program demonstrates several interesting mathematical properties:

1. **Multiplication patterns**: The 9 times table has beautiful symmetrical patterns
2. **Digital root magic**: Every multiple of 9, when you sum its digits repeatedly, always equals 9
3. **Divisibility shortcut**: You can quickly check if any number is divisible by 9 by summing its digits
4. **Cultural significance**: The number 9 appears in many contexts across different cultures

## Example Output

The program will show you:
- Complete multiplication table (9 × 1 through 9 × 10)
- Digital root calculations for each multiple
- Fun facts about the number 9
- Divisibility tests with various numbers

- Python 3.x
